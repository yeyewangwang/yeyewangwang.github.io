Personas: User Studies of An Interface

Context

For my UIUX class project, I worked on a project to understand the users of a popular interface. I chose McDonalds self-ordering kiosks because it is a widely used interface catering to a diverse set of customers.

An Introduction to the Interface

I noticed the kiosk as soon as I entered the McDonalds. A large touch screen dominates the kiosk; the users select menu items here.

After the user chooses the order, she would interact the following physical components in sequence:

  1. Card reader: when prompted, the user pays on the credit card reader
  2. Receipt slot: the user takes a receipt with an order number
  3. Table number holder: the user takes a table number holder, and type the number into the screen

Observations and Interviews

After observing the interface itself, I observed and interviewed 3 users who came into the McDonalds and used the kiosk. Below are the summaries of my observations:

Observation Summary

  1. Users 1 and 2 came in and went straight for the vending machines. They made their orders in 2 minutes and a minute respectively.
  2. Users 1 and 3 went back and forth on different pages in the menu, but user 2 directly made the order he wanted to make
  3. User 3 picked up a table number, looked at the cashiers, then put it back, and a minute later took a table number again
  4. User 3 sighed when he tried to scan in his membership with his cellphone but sighed when he could not successfully log in.
  5. User 1 clicked the button for paying at the cashier without hesitation; her actions were swift in general.
  6. Users 2 and 3 slowly took out their credit cards and paid on the checkout machine

Surprising fact: user 1 seems very familiar with checking out at McDonalds.

Questions

  1. How often do you visit this restaurant?
  2. How often do you use self check-out or vending machines in any stores?
  3. Why did you choose to eat here today?
  4. When you were using the kiosk, what are the things you liked or disliked the most?
  5. Would you add or remove a button to this machine? If so, which one and why?
  6. Why did you prefer this machine over the manned customer checkout?

Insights from Interview

  1. Two users came to the restaurant because they did not want to spend time cooking. One user said he's bored with home cooking'.
  2. Two users describe always preferring using the self checkout at all groceries or restaurants. One user says that he sometimes prefer talking to real person instead of a “robot”
  3. All users dislike one thing the most: they don’t know whether they should grab the table number or whether they should use the order number on the receipt for getting their order
  4. Two users mentioned removing the log-in button to improve the user interface. They did not use them anyways.
  5. Two users said they use self-checkout because it’s faster. One user said she preferred making choices on the hierarchical electronic menu rather than looking at the menu on the cashier wall, which lists all items in one view.

Conclusion
